virtuoso数据库配置

  • RDF
  • virtuoso
  • 数据库配置

virtuoso数据库的配置文件存放在/usr/local/virtuoso7/var/lib/virtuoso/db/目录下，virtuoso.ini文件，可以对virtuoso的数据保存位置和内存大小等信息进行配置。

常用配置项介绍：
[Database]
DatabaseFile = /data/virtuoso/virtuoso/db/virtuoso.db；数据库存放路径
ErrorLogFile = /data/virtuoso/virtuoso/db/virtuoso.log；日志存放路径
LockFile = /data/virtuoso/virtuoso/db/virtuoso.lck；锁
TransactionFile = /data/virtuoso/virtuoso/db/virtuoso.trx；事务
xa_persistent_file = /data/virtuoso/virtuoso/db/virtuoso.pxa；持久化

[Parameters]
ServerPort = 1111 ; 数据库端口号，用于tcp连接
MaxClientConnections = 300；最大连接数
DirsAllowed = .,/root/gcm, /data/virtuoso ；上传文件可以存放路径，上传数据时需要把数据文件放到该指定目录下
MaxQueryMem = 8G ；查询最大占用内存
NumberOfBuffers = 2040000 ；数据缓存块占用个数，
MaxDirtyBuffers = 1500000；最大脏数据缓存块个数
可根据机器内存进行相应设置，参考设置：
;; Uncomment next two lines if there is 32 GB system memory free
;NumberOfBuffers = 2720000
;MaxDirtyBuffers = 2000000
;; Uncomment next two lines if there is 48 GB system memory free
;NumberOfBuffers = 4000000
;MaxDirtyBuffers = 3000000
;; Uncomment next two lines if there is 64 GB system memory free
;NumberOfBuffers = 5450000
;MaxDirtyBuffers = 4000000

[HTTPServer]
ServerPort = 8890 ；http访问端口号
MaxClientConnections = 10；最大连接数
ServerThreads = 10
MaxKeepAlives = 10
KeepAliveTimeout = 10
MaxCachedProxyConnections = 10
Charset = UTF-8

[SPARQL];
ResultSetMaxRows = 10000 ; sparql查询时 返回最大三元组条数
