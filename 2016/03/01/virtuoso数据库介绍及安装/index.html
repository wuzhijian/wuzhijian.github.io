virtuoso数据库介绍及安装

  • RDF
  • virtuoso
  • 数据库安装

virtuoso数据库是一个高性能的rdf数据库，可支持百亿条三元组，支持sparql1.1语法查询。

一、开源版本安装包下载地址：

https://github.com/openlink/virtuoso-opensource/tree/stable/7

  1. 安装依赖：yum install gcc gmake autoconf automake libtool flex bison gperf gawk m4 make openssl-devel readline-devel wget
  2. 下载安装包：wget https://codeload.github.com/openlink/virtuoso-opensource/zip/stable/7
  3. 解压安装包：unzip 7
  4. 编译安装配置：cd virtuoso-opensource-stable-7/
    sh autogen.sh
    ./configure –prefix=/usr/local/virtuoso7 –datarootdir=/data/virtuoso –enable-demo-vad –enable-rdb2rdf-vad –enable-tutorial-vad–enable-sparqldemo-vad –enable-rdfmappers-vad –enable-isparql-vad –enable-ods-vad –with-readline –with-mone
    5.编译：nice make
    6.安装：make install

二、启动运行virtuoso：

cd /usr/local/virtuoso7/var/lib/virtuoso/db
nohup /usr/local/virtuoso7/bin/virtuoso-t -f &(注意：执行该启动命令时，需要先进入到上面的目录中)

三、停止virtuoso：

cd /usr/local/virtuoso7/bin
/usr/local/virtuoso7/bin/isql 1111 dba dba -K

四、virtuoso访问：

1.sparql查询地址：localhost:8890/sparql
2.web管理地址：localhost:8890/

五、上传数据到virtuoso：

上传xml格式数据: /usr/local/virtuoso7/bin/isql 1111 dba dba exec=”DB.DBA.RDF_LOAD_RDFXML_MT (file_to_string_output (‘/data/virtuoso/go_datas/go5.xml’), ‘’, ‘test’);”
上传nt格式数据: /usr/local/virtuoso7/bin/isql 1111 dba dba exec=”DB.DBA.TTLP_MT (file_to_string_output (‘/data/virtuoso/go_datas/go5.nt’), ‘’, ‘go’);”
